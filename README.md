# Wedding Website

A modern, responsive wedding website built with **Next.js** and **React**, featuring password-protected access, online RSVP management, Google Sheets integration, and a polished editorial-style user interface. The application is designed to provide guests with all essential wedding information while allowing the hosts to easily manage attendance and event details.

---

## Features

- Responsive design for desktop and mobile
- Password-protected guest access
- Online RSVP system
- Google Sheets integration for RSVP management
- Dynamic guest lookup
- Wedding details and event schedule
- Travel and lodging information
- Registry page
- FAQ section
- Wedding party page
- Our Story page
- Mobile navigation menu
- Elegant editorial-inspired user interface

## Project Highlights

### Secure Guest Access

- Password-protected website

### RSVP Management

- Search guests by name
- Record attendance responses
- Store RSVP status and notes in Google Sheets

## Running Locally

Clone the repository:

```bash
git clone https://github.com/yourusername/wedding-website.git
cd wedding-website
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
